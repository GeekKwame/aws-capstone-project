About MyCloudApp

MyCloudApp demonstrates a real-world AWS architecture: users reach the application through Amazon CloudFront, which terminates TLS and caches content at the edge. Dynamic requests flow through an Application Load Balancer to EC2 instances running Nginx, while heavy static assets are served directly from a private S3 bucket via CloudFront Origin Access Control.

Deployments are automated with GitHub Actions, pushing code changes to EC2 over SSH on every merge to main.
